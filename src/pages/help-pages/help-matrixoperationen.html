Auf dieser Seite können verschiedene Matrixoperationen berechnet werden.

  • M⁻¹ zur Matrixinversion
  • |M| zur Ausgabe der Determinante
  • Mᵀ zum Transponieren der Matrix

Hierzu muss zuerst die Input-Matrix eingegeben werden, auf der die Operationen durchgeführt werden sollen. Diese Matrix ist die obere der beiden Matrizen auf der Seite. Beim Klick auf einen der Buttons wird die entsprechende Operation durchgeführt und berechnet. Das Ergebnis der Berechnung wird in der unteren Matrix ausgegeben.

Wichtig zu beachten ist, dass die Berechnung der Matrixinversion und der Determinante nur bei quadratischen Matrizen möglich ist. Eine quadratische Matrix ist eine Matrix, bei der die Anzahl an Zeilen und Spalten identisch ist.

Das Ergebnis der Matrixinversion ist die Inverse der Matrix. Die Inverse einer Matrix ist so definiert, dass das Ergebnis einer Multiplikation der Inversen mit der ursprünglichen Matrix die Einheitsmatrix ergibt. Es gibt jedoch nicht für jede Matrix eine Inverse. Beispiele hierfür sind Matrizen, die in einer Zeile oder Spalte ausschließlich Nullen enthalten, oder Matrizen mit linearen Abhängigkeiten.

Die Determinante ist ein Skalar und wird einer quadratischen Matrix zugeordnet. Sie wird aus den Einträgen der Matrix berechnet. Die Determinante kann beipielsweise als Hilfsmittel für die Lösung eines linearen Gleichungssystems genutzt werden.

Das Ergebnis des Transponierens ist die transponierte oder auch gespiegelte Matrix. Beim Transponieren werden die Zeilen und Spalten vertauscht (d.h. die Matrix an der Hauptdiagonalen gespiegelt). Die erste Zeile entspricht bei der transponierten Matrix der ersten Spalte, die zweite Zeile entspricht bei der transponierten Matrix der zweiten Spalte, usw.

Die Matrizen können über den Button 'Leeren' geleert werden. Die Werte der Zellen werden dabei auf 0 gesetzt. Die Größe der Eingabematrix wird auf 3x3 zurückgesetzt.
