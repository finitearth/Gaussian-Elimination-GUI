Die Inverse einer Matrix wird in der linearen Algebra unter anderem bei der Lösung linearer Gleichungssysteme verwendet. Die inverse Matrix ist die Matrix, die multipliziert mit der ursprünglichen Matrix die Einheitsmatrix ergibt. Die Einheitsmatrix oder auch Identitätsmatrix ist eine quadratische Matrix, deren Elemente auf der Hauptdiagonalen eins und überall sonst null sind. Zum Berechnen der Inversen bietet sich unter anderem der Gauß-Algorithmus an.

Diese Seite kann für die Berechnung der Inversen, aber auch zur Gleichung zweier Matrizen genutzt werden.

Zuerst wird die Dimension spezifiziert. Da es sich bei der Einheitsmatrix um eine quadratische Matrix handelt, die Anzahl an Zeilen also identisch zur Anzahl der Spalten ist, kann hier nur die gesamte Dimension angepasst werden. Hierfür kann die entsprechende Zahl direkt in das Textfeld eingegeben, oder die Zahl über die Pfeile rechts angepasst werden. Hier gilt die Mindest- und Maximalgröße von 2 bzw. 9. Nach der Anpassung der Dimensionen kann mit den Zeilenoperationen begonnen werden. Über die Pfeile neben der rechten Matrix Über die Pfeile neben den rechten Seiten können die entsprechenden Dropdowns und Textfelder ein- und ausgeklappt werden, um die Zeilenoperationen einzugeben. Alle Koeffizienten einer Zeile können mit einem Skalar multipliziert, dividiert, addiert oder subtrahiert werden. Die entsprechenden Operatoren können im ersten und zweiten Dropdown ausgewählt werden. Für die Felder der Zeilenoperationen gilt ebenfalls, dass ein leeres Feld eine Null bedeutet. Wenn die Koeffizienten also nicht mit einem Skalar multipliziert oder dividiert werden sollen, kann das erste Textfeld leer bleiben. Das gleiche gilt für das Addieren und Subtrahieren von Werten im zweiten Textfeld. Über die letzten beiden Dropdowns kann eine Verrechnung der Zeile mit einer anderen Zeile durchgeführt werden, wobei die Zahlen den Beschriftungen der Zeilen auf der Seite entsprechen. Sobald alle Werte eingegeben wurden, kann die Berechnung über den Button "Berechnen" durchgeführt werden. Das Ergebnis der Berechnung wird in den unteren Matrizen ausgegeben. Liefert das Ergebnis der Berechnung ein zufriedenstellendes Ergebnis, können die Werte mit dem Button "Hoch" nach oben übertragen werden, sodass mit dem Ergebnis weitergerechnet werden kann. Sollte das Ergebnis nicht den Erwartungen entsprechen, können die zuvor eingegebenen Werte abgeändert und das Ergebnis neu berechnet werden, so lange bis das gewünschte Ergebnis herauskommt.

Sollte es Probleme beim schrittweisen Lösen geben, kann die Lösung stattdessen auch vom System berechnet werden. Dazu wird der Button "Lösen" verwendet. Die Lösung wird dann direkt in den unteren Matrizen ausgegeben. Um die Berechnung der Inversen zu üben kann über den Button "Aufgabe erstellen" eine zufällige Aufgabe kreiert werden. Die Aufgabe wird automatisch an die angegebene Dimension angepasst. Auch auf dieser Seite kann über den Toggle-Button zwischen Dezimalzahlen und Bruchzahlen gewechselt werden. Sollten bereits Werte in den Matrizen stehen, werden diese automatisch umgerechnet und abgeändert. Um die Werte auf Null und die Matrizen zurückzusetzen kann der Button "Leeren" verwendet werden.
