Der Gauß-Algorithmus, auch bekannt als Gaußsches Eliminationsverfahren, ist ein Verfahren zur Lösung linearer Gleichungssysteme. Dabei wird das Additionsverfahren auf die erweiterte Koeffizientenmatrix angewendet. Die Koeffizientenmatrix ist eine Matrix, die aus den Koeffizienten, also den Zahlen vor den Unbekannten, eines linearen Gleichungssystems besteht. Sie ist ein Hilfsmittel, um lineare Gleichungssysteme unter anderem mit dem Gauß-Algorithmus zu lösen. Die Koeffizientenmatrix fasst die Koeffizienten auf den linken Seiten der Gleichungen zu einer Matrix zusammen. Das Additionsverfahren ist ein Verfahren, bei dem die Gleichungen so addiert werden, dass Variablen wegfallen bzw. die Koeffizienten Null werden. Ziel des Additionsverfahrens ist es, eine Gleichung mit einer Variablen zu erhalten, die dann gelöst werden kann. Das Additionsverfahren basiert auf der Erkenntnis, dass die Summe der linken Seiten der Gleichungen gleich der Summe der rechten Seiten ist. Um das Gleichungssystem zu lösen, muss die Koeffizientenmatrix so umgeformt werden, dass unter der Diagonalen nur noch Nullen stehen, dies wird auch Stufenform genannt. Diese Seite kann genutzt werden, um ein Gleichungssystem schrittweise mit Hilfe von Zeilenoperationen mit dem Gauß-Algorithmus zu lösen. Zuerst wird die Anzahl Gleichungen, die Anzahl Variablen und die Anzahl der rechte Seiten spezifiziert. Die Anzahl Gleichungen bestimmt die Anzahl an Zeilen und die Anzahl Variablen bestimmt die Anzahl an Spalten. Danach können die entsprechenden Werte in die Koeffizientenmatrix und die rechte Seite eingegeben werden.

Um die Stufenform zu erreichen, müssen Zeilenoperationen durchgeführt werden. Über die Pfeile neben den rechten Seiten können die entsprechenden Dropdowns und Textfelder ein- und ausgeklappt werden, um die Zeilenoperationen einzugeben. Alle Koeffizienten einer Zeile können mit einem Skalar multipliziert oder durch ein Skalar dividiert werden. Der entsprechende Operator kann im ersten Dropdown ausgewählt werden. Im zweiten und dritten Dropdown kann die Zeile um eine andere Zeile addiert oder subtrahiert werden, wobei die Zahlen in Klammern den Beshriftungen der Zeilen auf der Seite entsprechen. Die Zeile, mit der die aktuelle Gleichung verrechnet werden soll, kann außerdem mit einem beliebigen Wert multipliziert oder um einen beliebigen Wert dividiert werden. Sobald alle Werte eingegeben wurden, kann die Berechnung über den Button "Berechnen" durchgeführt werden. Das Ergebnis der Berechnung wird in der unteren Matrix ausgegeben. Liefert das Ergebnis der Berechnung ein zufriedenstellendes Ergebnis, können die Werte mit dem Button "Hoch" nach oben übertragen werden, sodass mit dem Ergebnis weitergerechnet werden kann. Sollte das Ergebnis nicht den Erwartungen entsprechen, können die zuvor eingegebenen Werte abgeändert und das Ergebnis neu berechnet werden, so lange bis das gewünschte Ergebnis herauskommt.

Sobald die Stufenform erreicht ist, kann durch Rückwärtseinsetzten die Lösung berechnet werden. Das heißt, es wird mit der letzten Zeile begonnen und diese nach der Unbekannten aufgelöst. Diese Lösung wird in die vorherige Zeile eingesetzt, die wiederum nach der nächsten Unbekannten aufgelöst wird. Dies wird so lange wiederholt, bis alle Unbekannten bestimmt sind. Ein lineares Gleichungssystem kann entweder keine, eine oder unendlich viele Lösungen haben. Eine Lösung bedeutet, dass sich die Graphen in einem Punk schneiden, der entsprechend berechenbar ist. Wenn die Graphen sich nicht schneiden, zum Beispiel weil sie parallel sind, dann hat das lineare Gleichungssystem keine Lösung. Unendlich viele Lösungen hat es dann, wenn die Graphen identisch verlaufen.

Sollte es Probleme beim schrittweisen Lösen des Gleichungssystems geben, kann die Lösung stattdessen auch vom System berechnet werden. Dazu wird der Button "Lösen" verwendet. Die Lösung wird dann direkt in der unteren Matrix ausgegeben. Um das Lösen linearen Gleichungssysteme zu üben kann über den Button "Aufgabe erstellen" eine zufällige Aufgabe erzeugt werden. Die Aufgabe wird automatisch an die angegebene Anzahl Gleichungen, Variablen und rechte Seiten angepasst. Auch auf dieser Seite kann über den Toggle-Button zwischen Dezimalzahlen und Bruchzahlen gewechselt werden. Sollten bereits Werte in den Matrizen oder dem Vektor stehen, werden diese automatisch umgerechnet und abgeändert. Um die Werte in der Matrix auf Null zurückzusetzen kann der Button "Leeren" verwendet werden.
