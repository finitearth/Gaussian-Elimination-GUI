Hilfe - Matrixoperationen

Bedienungsanleitung & Tipps

Auf dieser Seite können verschiedene Matrixoperationen berechnet werden.

  • M⁻¹ zur Matrixinversion
  • |M| zur Ausgabe der Determinante
  • Mᵀ zum Transponieren der Matrix

Hierzu muss zuerst die Input-Matrix eingegeben werden, auf der die Operationen durchgeführt werden sollen. Diese Matrix ist die obere der beiden Matrizen auf der Seite. Bei Klick auf einen der Buttons wird die entsprechende Operation durchgeführt und berechnet. Das Ergebnis der Berechnung wird in der unteren Matrix im Tab 'Ergebnis' ausgegeben.

Wichtig zu beachten ist, dass die Berechnung der Matrixinversion und der Determinante nur bei quadratischen Matrizen möglich ist. Eine quadratische Matrize ist eine Matrize, bei der die Anzahl an Zeilen und Spalten identisch ist.

Das Ergebnis der Matrixinversion ist die quadratische inverse Matrix. Das Ergebnis einer Multiplikation einer Matrix mit ihrer Inversen ist die Einheitsmatrix. Es gibt jedoch nicht für jede Matrix eine Inverse. Ein Beispiel hierfür ist eine Matrix, die in einer Zeile oder eine Spalte ausschließlich aus Nullen besteht oder auch eine Matrix linearen Abhängigkeiten.

Die Determinante ist eine Zahl bzw. ein Skalar und wird einer quadratischen Matrix zugeordnet. Sie wird aus den Einträgen der Matrix berechnet. Die Berechnung der Determinante kann als Hilfsmittel für die Lösung des linearen Gleichungssystems genutzt werden.

Das Ergebnis des Transponierens ist die transponierte oder auch gespiegelte Matrix. Beim Transponieren werden die Zeilen und Spalten vertauscht. Die erste Zeile entspricht bei der transponierten Matrix der ersten Spalte, die zweite Zeile entspricht bei der transponierten Matrix der zweiten Spalte, usw.
