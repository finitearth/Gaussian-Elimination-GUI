Allgemein

Diese Seite dient der Beschreibung, wie die Website zu nutzen ist. Auf diese Seite kann auf jeder Unterseite über den Link im Menü in der oberen rechten Ecke über ⓘ zugegriffen werden. Weitere Informationen wie mathematische Hintergründe und Tipps sind jeweils auf den Unterseiten passend zur jeweiligen Themenstellung zu finden.

Die Startseite dient als Einstiegspunkt auf die Website. Neben dem Navigationsmenü, dass auf jeder Seite zur Verfügung steht, beinhaltet die Startseite Links zu allen Unterseiten. Im unteren rechten Eck ist auf jeder Seite ein Link zum Impressum angegeben.

Jede Unterseite verfügt über einen Hilfe-Button oben rechts. Durch Drücken des Buttons öffnet sich ein Fenster, welches Tipps und Hilfestellung passend zum Thema der Seite beinhaltet. Außerdem beinhaltet das Fenster nähere Informationen zur Bedienung bzw. Verwendung der Seite, unter anderem mit Erklärung der Buttons.

Allgemein gilt, dass die Dimension der Matrizen flexibel anpassbar ist, die Matrix muss jedoch mindestens 2x2 bzw. darf maximal 9x9 groß sein. Kleinere oder größere Matrizen sind nicht möglich. Ist eine Zelle einer Matrix nicht ausgefüllt, so wird dieses automatisch mit dem Wert 0 belegt und erkannt.

Die Angabe der Zellenwerte innerhalb einer Matrix erfolgt entweder als Dezimalzahlen oder als Brüche. Auf jeder Unterseite steht ein Toggle-Button zur Verfügung, um von Dezimalzahlen auf Brüche und von Brüche auf Dezimalzahlen umzustellen.

Matrizenrechnung

Auf der Unterseite Matrizenrechnung können bis zu 26 Matrizen erstellt und Berechnungen darauf ausgeführt werden. Standardmäßig beinhaltet die Seite 2 Input-Matrizen. Mit dem Button '+ Matrix hinzufügen' wird eine weitere Matrix hinzugefügt, bis das Limit von 26 Matrizen erreicht ist. Bei jeder Matrix kann die Anzahl an Spalten und Zeilen flexibel angepasst werden, wobei auch hier die Mindest- und Maximalgrenzen von 2 bzw. 9 gelten. Um die Anzahl an Zeilen und Spalten anzupassen müssen die '+'- und '-'-Buttons im oberen linken Eck verwendet werdne. Im Seitenabschnitt 'Berechnung' steht ein Freitextfeld zur Verfügung, in welchem die durchzuführene Berechnung angegeben werden kann. Um die Berechnung durchführen zu können, muss sie korrekt sein. Um sicherzustellen, dass eine korrekte Berechnung erfolgt sind folgende Objekte und Operationen zu verwenden:

  • die Matrizen selbst - diese werden angesprochen über den jeweiligen Buchstaben
  • ein + für die Addition von Matrizen
  • ein - für die Subtraktion von Matrizen
  • ein * für die Multiplikation von Matrizen
  • eine Zahl und ein * für das Skalarprodukt
  • ein x für das Kreuzprodukt von Matrizen
  • sowie Klammern, um die Reihenfolge der Operationen zu bestimmen
Nach Eingabe der Berechnung kann die Berechnung, wenn sie korrekt ist, über den Button 'Berechnen' ausgeführt werden. Das Ergebnis wird in der Matrix neben dem Freitextfeld angezeigt.

Matrixoperationen

Die Unterseite Matrixoperationen bietet die Möglichkeit, die Inverse, die Determinante oder die transponierte Matrix einer Matrix ausgeben zu lassen. Die Anzahl an Zeilen und Spalten auf dieser Unterseite kann auf zwei Arten angepasst werden. Zum einen können die '+'- und '-'-Buttons im oberen linken Eck der Input-Matrix genutzt werden. Zum anderen können die Textboxen im oberen linken Bereich der Seite direkt unter der Seitenüberschrift genutzt werden. Die gewünschte Anzahl kann entweder direkt in die Textbox eingetippt werden oder der Wert kann über die Pfeile am rechten Rand der Textboxen inkrementiert bzw. dekrementiert werden. Für die durchzuführenden Matrixoperationen stehen drei Buttons unterhalb der Eingabematrix zur Verfügung:

  • M⁻¹ zur Matrixinversion
  • |M| zur Ausgabe der Determinante
  • Mᵀ zum Transponieren der Matrix
Unter dem Tab Ergebnis wird das Ergebnis der Operation ausgegeben.

Gl-System

Die Unterseite Gl-System kannst genutzt werden, um Gleichungssysteme schrittweise selbst zu lösen oder um die Lösung sofort zu erhalten. Zuerst wird im ersten Abschnitt die Anzahl an Gleichungen (entspricht Anzahl der Zeilen), die Anzahl der Variablen (entspricht Anzahl der Spalten) und Anzahl rechte Seiten ausgewählt. Die entsprechende Anzahl kann direkt im Feld angeben oder über die Pfeile abgeändert werde. Danach folgen die Eingabefelder für die Gleichungen. Rechts davon befindet sich pro Zeile ein Button mit einem Pfeil nach rechts. Wenn dieser betätigt wird, erscheint eine Combo-Box mit Drop-Down- und Eingabe- Felden. Mithilfe dieser können die Zeilenoperationen, die durchgeführt werden sollen, zusammengestellt werden. Die Comob-Box kann, falls nicht benötigt, über den Button mit dem Pfeil nach links wieder eingeklappt werden. Falls Hilfe beim Lösen des Gleichungssystems benötigt wird, kann über den Button 'Tipp' ein entsprechender Tipp ausgegeben oder das Gleichungssystem über den Button 'Lösen' sofort gelöst werden. Wenn die auszuführenden Zeilenoperationen eingegeben wurden, können dieser über den Button 'Berechnen' durchgeführt werden. Der Button gibt das Ergebnis der Verrechnung der Matrix mit den Zeilenoperationen darunter im Tab 'Berechnung' aus. Wenn das Ergebnis zufriedenstellend ist, kann es über den Button 'Hoch' auf die Eingabematrix übertragen werden, um damit weiterzurechnen. Im Tab 'Historie' kannst der gesamte Verlauf der Zeilenoperationen schrittweise angesehen und ausgedruckt oder, falls notwendig, die Historie geleert werden.

Inversenberechnung

Die Unterseite Inversenberechnung kannst nicht nur zur Berechnung der Inverse einer Matrix, sondern auch zur Gleichung zweier Matrizen genutzt werden. Die Dimension der beiden Matrizen werden direkt im ersten Abschnitt auf der Seite angegeben. Wie auf den anderen Seiten handelt es sich dabei um ein Textfeld, in das die Anzahl der Dimensionen direkt eingetragen oder über die Pfeile verändert werden kann. Danach folgen die Eingabefelder für die beiden Matrizen. Die rechte Matrix ist vorbelegt mit der Einheitsmatrix zur Berechnung der Inversen. Die Werte lassen sich überschreiben, falls nicht die Inverse berechnet, sondern eine Matrizengleichung gelöst werden soll. Wie auf der Unterseite Gl-System lösen kannst hier schrittweise über Zeilenoperationen zur Lösung gelant werden. Die Eingabe der Zeilenoperationen erfolgt über die Combo-Boxen, die über die Buttons mit den Pfeilen nach rechts bzw. links ein- und ausgeklappt werden können. Außerdem kann über den Buttons 'Tipp' ein Tipp bzw. über den Button 'Lösen' direkt die Lösung ausgeben lassen. Nach Eingabe der Zeilenoperationen kann die Berechnung über den Button 'Berechnen' durchgeführt werden. Das Ergebnis wird in dem Tab 'Berechnen' angezeigt. Wenn das Ergebnis zufriedenstellend ist und damit weitergerechnet werden soll, kann es über den Button 'Hoch' in die Eingabematrizen übertragen werden. Im Tab 'Historie' kann der gesamte Verlauf der Zeilenoperationen schrittweise angesehen und ausgedruckt oder, falls notwendig, die Historie geleert werden.
