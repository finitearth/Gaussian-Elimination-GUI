getDeterminant() → {Fraction}

Computes the determinant of the matrix recursively. If the matrix has dimensions 1x1, then the determinant is simply the only element of the matrix. Otherwise, the determinant is calculated as the sum of the products of the elements of the first row by the determinants of their corresponding submatrices. The sign of each product alternates depending on the sum of the indices of the row and column of the element.

mul(other) → {Matrix}

Multiplies this matrix by the input value, which can be either a scalar value represented as a Fraction or a Matrix object. If the input is a scalar value, the function returns a new Matrix object where each cell is multiplied by the scalar value. If the input is a Matrix object, the function returns a new Matrix object that is the product of the two matrices. Throws an error if the input is not a scalar value or a Matrix object.
